# Truss-Bridge-Model-Vibration-Benchmark-Dataset
This dataset provides benchmark vibration data of a truss bridge model, including dynamic response measurements under multiple structural damage scenarios and various excitation conditions. It can be used for modal identification, signal processing, moving load identification, and anomaly detection.

The dataset will be made publicly available upon publication of the paper. For early access to the data prior to publication, please contact zhuyk@zju.edu.cn or hpwan@zju.edu.cn
